## 简介 (Introduction)

SheetMetalAI 是一款基于生成式 AI (Volcengine Doubao) 的智能钣金加工辅助工具。它能够自动分析钣金图纸（图片、PDF、DXF），提取关键几何参数，并实时生成 3D 折弯预览和 2D 展开图，极大地简化了从设计到加工的流程。

**主要功能:**
*   📄 **智能图纸识别:** 支持上传 JPG, PNG, PDF, DXF。
*   🤖 **AI 参数提取:** 自动识别长宽、折弯深度、板厚等参数。
*   🧊 **3D 实时预览:** 可交互的折弯件 3D 模型。
*   📐 **2D 自动展开:** 生成带折弯线的平面展开图。
*   🛠️ **参数化设计:** 支持手动微调识别结果，实时更新视图。

## 本地运行 (Run Locally)

**前置条件:** Node.js (v18+)

1.  **安装依赖:**
    ```bash
    npm install
    ```

2.  **配置环境变量:**
    创建一个 `.env` 文件，并填入你的 Gemini API Key:
    ```env
    GEMINI_API_KEY=your_api_key_here
    ```

3.  **启动开发服务器:**
    ```bash
    npm run dev
    ```
